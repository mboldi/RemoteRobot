# RemoteRobot

Futószalagon érkező termékek félautomata selejtdetekciója robotkar és VR eszközök segítségével, manuális vezérlési lehetőséggel.

## Használt technológiák:

- ROS kinetic
- MoveIt
- OpenCV
- Unity
- HTC Vive
